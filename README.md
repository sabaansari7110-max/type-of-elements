# Type-Of-Elements

## Description
This project provides a simple and clear overview of python data along with their class representations(e.g., <class "int">. It is designed to help beginners understand how different types of data are categorixed and handled in python.

## Numeric types
**<class 'int'>** integer(e.g. 10)
<br>
**<class 'float'>** decimal (e.g. 10.5)
<br>

## Text Type
**<class 'str>** string (e.g. "HELLO)

## Boolean Type
**<class 'bool'>** True/ False
